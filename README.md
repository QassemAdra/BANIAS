# Qassem Platform

Qassem Platform is an open-source archive platform for documenting historical events.

It includes:

• Qassem Framework (WordPress theme framework)
• Qassem Archive Core (archive platform plugin)
• Qassem Starter Sites (demo importer)

Features:

- Evidence archive
- Map visualization
- Timeline system
- AI evidence assistant
- Cloudflare R2 media storage
- RTL-first design
- Fully responsive UI

Project Structure:

qassem-framework/
WordPress theme framework

qassem-archive-core-pro/
Core archive platform plugin
