Project: Qassem Archive

WordPress plugin + theme for a public evidence archive.

Structure:
- qassem-archive-core-pro → main plugin
- qassem-framework → main theme

Requirements:
- WordPress 6.9
- PHP 8.2
- Must activate without fatal errors

Features:
- Evidence archive
- Map (Leaflet)
- Timeline
- AI evidence assistant
- Cloudflare R2 media storage

Coding rules:
- class-* file naming
- namespaces QA\
- WordPress hooks and APIs
- sanitize/escape output
