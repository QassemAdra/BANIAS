# Qassem Platform Architecture

This project contains two main components:

1. Qassem Framework
   WordPress theme framework responsible for UI and layout.

2. Qassem Archive Core
   WordPress plugin responsible for archive functionality.

Architecture principles:

- Modular code
- No monolithic files
- Clean namespace separation
- REST-first data access
- Frontend components separated from backend logic

Target code size:
50k+ lines across modules.
